DecisionHelperBot

  • This is a simple app that prompts the user for questions about decisions they are having trouble making.
  • The app then provides a response telling the user if it is a good decision, a bad decision, or unclear.
  • In addition, the user can provide feedback as to whether the response was helpful
  • When writing the user interface, I did some additional experimentation with fonts, background colors, and item layouts.
